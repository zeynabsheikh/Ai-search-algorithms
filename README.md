# AI Search Algorithms Project

This project demonstrates BFS, DFS, and A* search algorithms with GUI visualization using Python and Pygame.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```
Create and activate virtual environment:
Windows PowerShell:
```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Windows CMD:
```bash
python -m venv .venv
.\.venv\Scripts\activate.bat
```

Linux/macOS:
```bash
python -m venv .venv
source .venv/bin/activate
```

install dependencies:
```bash
pip install -r requirements.txt
```

(Make sure pygame is listed in requirements.txt)

Running the Project:
```bash
python main.py
```

Features

BFS, DFS, and A* search algorithms

Interactive GUI for visualization

Shows step-by-step exploration of search

Best case and worst case scenario visualizations
